### 3.3 Proyecto Ejemplo: Laguna de Tratamiento Minero

```
LAGUNA DE TRATAMIENTO DE AGUAS DE PROCESO
Minería / Sonora, México

─────────────────────────────────────────────────────

EL DESAFÍO

Diseñar e instalar un sistema de impermeabilización para
una laguna de tratamiento de 12,000 m² en zona de alta
exposición UV y variación térmica extrema.

Requerimiento crítico: cero infiltraciones en vida útil
proyectada de 25 años bajo condiciones químicas agresivas.

LA SOLUCIÓN

Sistema de doble liner con geomembrana HDPE 2.0mm +
geocompuesto drenante para detección de fugas.

Diseño de anclajes perimetrales considerando expansión
térmica y asentamientos diferenciales proyectados.

ALCANCE EJECUTADO

→ Preparación y certificación de subrasante
→ Instalación de geocompuesto drenante (12,500 m²)
→ Instalación de geomembrana HDPE 2.0mm (12,800 m²)
→ Sistema de detección de fugas con sumidero central
→ Soldadura por termofusión, 100% probada con vacío
→ Documentación completa y certificado de instalación

─────────────────────────────────────────────────────

DATOS TÉCNICOS

Superficie              Material Principal       Plazo Ejecución
12,800 m²               HDPE 2.0mm              38 días

Juntas Soldadas         Pruebas de Vacío        Temperatura Trabajo
2,400 ml                100%                    -5°C a 42°C
```
